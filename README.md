CRUD - Gestion de Vehiculos

Resumen:
Esta aplicacion permite gestionar distintos tipos de vehiculos (Auto, Moto y Camion) mediante operaciones CRUD completas:
Agregar, Listar y Eliminar vehiculos
Ordenar, Filtrar
Guardar y cargar datos en formato CSV y JSON
La interfaz grafica fue desarrollada con JavaFX y permite realizar todas las operaciones desde botones y campos de texto.
